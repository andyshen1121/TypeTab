# Chrome Web Store Listing - 中文

## 名称
TypeTab

## 简短描述（132 字符以内）
快速标签页搜索和智能重复检测。Spotlight 风格搜索，一键找到目标页面。

## 详细描述

打开了几十个标签页？TypeTab 帮你高效管理。

**Spotlight 搜索**
按 Ctrl+Shift+K（Mac: Cmd+Shift+K）唤起搜索，按标题或 URL 即时过滤所有已打开的标签页。方向键导航，Enter 切换，跨窗口搜索。

**重复标签页检测**
当你打开一个已经存在的页面时，TypeTab 即时拦截：
- 提示模式：弹出提示条，询问是否切换到已有标签页
- 静默模式：自动切换到已有标签页，重复页面根本不会加载

**点击图标即可设置**
点击工具栏 TypeTab 图标，搜索和设置一键切换：
- 开启/关闭重复检测
- 切换提示模式和静默模式
- 支持 URL 精确匹配和域名匹配

**隐私优先**
TypeTab 完全在本地运行，不采集任何数据，不连接外部服务器。所有标签页信息留在你的浏览器内。

**键盘驱动**
为键盘优先的工作流设计。一个快捷键搜索，方向键导航，Enter 切换，Esc 关闭。
